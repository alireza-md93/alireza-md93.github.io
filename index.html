Featured Projects

GPU-Accelerated Viterbi Decoder

High-performance 100 Gb/s CUDA implementation of the Viterbi algorithm with an Object-oriented design and a user-friendly API.

  • Novel State-Exchange algorithm leveraging warp-level programming to minimize synchronization overhead and maximize L2 cache utilization
  • Persistent kernel design minimizing intermediate memory footprint
  • Multi-format support: 1-bit, 4-bit, 8-bit and floating-point
  • Dual processing cores: integer core with DPX SIMD intrinsics and floating-point core with half2 SIMD intrinsics

MIPS Processor Simulation

A comprehensive Verilog simulation of a simplified MIPS processor featuring advanced architectural capabilities accompanied by a python assembler script.

  • Pipelined multi-cycle architecture with comprehensive hazard handling (data, control, and structural)
  • Von Neumann architecture featuring unified instruction and data memory
  • Efficient conflict resolution between instruction fetch and data access stages
  • Complete programming guide and sample assembly code for the insertion sort algorithm

Travelling Salesman Problem (TSP) Solver

A high-performance Python library solving the Traveling Salesman Problem (TSP) with performance optimizations using parallel computing and machine-learning techniques.

  • Branch pruning based on the best-known (last minimum) tour cost to aggressively reduce search space
  • ML-guided branch ordering: Random Forest and Neural Network models trained to prioritize promising branches
  • Single-core and multi-core solver implementations
  • Full report documenting algorithms, training methodology, experimental setup, and performance results

N-Body Gravity Simulation

Interactive Python-based simulation of gravitational dynamics in an N-body system with customizable parameters.

  • Random body generation with customizable mass and size ranges
  • Real-time animated visualization of gravitational interactions
  • Color-coded visualization based on body mass
  • Flexible simulation parameters for experimentation
